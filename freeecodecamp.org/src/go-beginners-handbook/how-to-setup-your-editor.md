I recommend using [**<VPIcon icon="iconfont icon-vscode"/>Visual Studio Code**](https://code.visualstudio.com/) (aka VS Code) as your editor.

Read [<VPIcon icon="iconfont icon-vscode"/>Go in Visual Studio Code](https://code.visualstudio.com/docs/languages/go) for a quick “up and running” setup. At the bare minimum, install [<VPIcon icon="iconfont icon-vscode"/>the Go extension](https://marketplace.visualstudio.com/items?itemName=golang.go).
